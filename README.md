# Breast Cancer Prediction

A simple machine learning project that helps doctors figure out whether a tumor is **cancerous (Malignant)** or **safe (Benign)** — just from scan measurements.



---

## What it does

Takes 30 measurements from a breast scan and predicts:
- **Malignant** — the tumor is cancerous
- **Benign** — the tumor is not cancerous

---

## How to run

```bash
pip install scikit-learn
python breast_cancer.py
```

---

## Results

The model hits **97.37% accuracy** on test data.

Out of 114 test cases, it only got 3 wrong.

---

## Tech used

- Python
- scikit-learn (dataset + model)
- Logistic Regression
